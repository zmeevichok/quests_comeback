# Исправление деплоя на Vercel

## 🔍 Проблема:
Vercel не может найти файл `/src/main.tsx` и сборка падает.

## 🔧 Решение:

### 1. Проверь локальную сборку:
```bash
npm run build
```

### 2. Если сборка падает локально:
- Проверь, что все файлы на месте
- Убедись, что `package.json` правильный
- Проверь, что `vite.config.ts` настроен

### 3. Если сборка работает локально:
- Проблема в настройках Vercel
- Попробуй пересоздать проект на Vercel
- Или используй другой способ деплоя

## 🚀 Альтернативные способы деплоя:

### Способ 1: Netlify
1. Зайди на netlify.com
2. Подключи GitHub репозиторий
3. Настройки: Build command: `npm run build`, Publish directory: `dist`

### Способ 2: GitHub Pages
1. В настройках репозитория включи Pages
2. Используй GitHub Actions для деплоя

### Способ 3: Firebase Hosting
1. Установи Firebase CLI
2. `firebase init hosting`
3. `firebase deploy`

## ✅ Рекомендация:
Попробуй Netlify - он лучше работает с Vite проектами.
