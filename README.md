# Quests Web

Веб-приложение для управления квестами с системой опыта.

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev-сервер:
```bash
npm run dev
```

3. Откройте браузер по адресу http://localhost:5173

## Функциональность

- **Доступные квесты**: Список всех доступных квестов с кнопкой "Взять"
- **Активные квесты**: Квесты в процессе выполнения с кнопкой "Завершить"
- **Завершенные квесты**: Завершенные квесты с галочкой
- **Профиль пользователя**: Отображение имени и баланса EXP
- **Система EXP**: Баланс увеличивается при завершении квестов
- **Скролл**: Все секции поддерживают прокрутку при большом количестве квестов
