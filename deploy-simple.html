🚀 Деплой Quests Web на Vercel

1. Подготовка

Убедись, что у тебя есть аккаунт на vercel.com

2. Создание репозитория на GitHub

1. Зайди на github.com

2. Создай новый репозиторий "quests-web"

3. Загрузи все файлы проекта

3. Деплой через Vercel

1. Зайди на vercel.com

2. Нажми "New Project"

3. Подключи GitHub репозиторий

4. Выбери папку с проектом

5. Нажми "Deploy"

4. Настройка Google OAuth

После деплоя получишь URL вида: https://quests-web-abc123.vercel.app

В Google Console добавь:

Authorized redirect URIs:
https://your-url.vercel.app/google-callback

Authorized JavaScript origins:
https://your-url.vercel.app

5. Альтернативный способ - через Vercel CLI

Если npm работает, попробуй:

npm install -g vercel
vercel login
vercel

✅ Готово!

После деплоя OAuth будет работать с реальным доменом
